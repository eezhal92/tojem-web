# tojem-web

## Kebutuhan Sistem
- Node JS 8.9.0+
- MySql
- Yarn 1.2.1+

## Cara Setup di Local
- Buat database di MySQL
- Buat file `.env`. Lihat contoh `.env.example`
- Import file `tojem-web-dump.sql` ke database tersebut
- Install dependencies `yarn install`
- Jalankan aplikasi `yarn run dev`
